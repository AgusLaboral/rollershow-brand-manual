# Referencias fotográficas

Estas imágenes calibran la dirección visual. No son stock libre ni una promesa de que cualquier tercero pueda reutilizarlas en campañas.

- `blackout-gris-bedroom-reference.jpg`: ambiente real, producto completo y luz cálida.
- `sunscreen-living-bright-reference.jpg`: filtrado de luz, vista y espacio negativo.

Antes de publicar una fotografía se registra origen, fecha, producto, locación, autorización y vencimiento de uso. Si falta uno de esos datos, la imagen no se entrega a producción.
