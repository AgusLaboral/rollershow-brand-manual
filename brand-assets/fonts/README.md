# Tipografías

## Bricolage Grotesque

- Uso: títulos, cifras grandes y momentos de marca.
- Ejes aprobados: `opsz 12–96`, `wght 400–800`.
- Archivo web incluido: `bricolage-grotesque-latin-variable.woff2`.
- Procedencia: https://fonts.google.com/specimen/Bricolage+Grotesque
- Licencia: SIL Open Font License 1.1.

## Manrope

- Uso: texto, formularios, botones, tablas y documentos.
- Pesos aprobados: 400, 500, 600 y 700.
- Archivo web incluido: `manrope-latin-variable.woff2`.
- Procedencia: https://fonts.google.com/specimen/Manrope
- Licencia: SIL Open Font License 1.1.

Los archivos OFL de ambas familias están incluidos en esta carpeta. El manual publicado carga estas copias locales y no depende de Google Fonts en tiempo de ejecución. Para imprenta se incrustan las fuentes en el PDF.
